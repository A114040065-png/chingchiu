# 🗺️ 跨城市美食互動導覽網頁（期末專案）

這是一個專為年輕人打造、具備活潑美式潮流視覺風格的**多頁式響應式（RWD）美食導覽網站**。專案中收錄了新竹、台南、嘉義三個城市各 5 家指標性排隊美食，並整合了實用的店家資訊與 Google 地圖導航功能。

## 🛠️ 專案亮點與技術特點

1. **多網頁路由架構**：專案包含三個獨立網頁（`index.html`、`tainan.html`、`chiayi.html`），透過上方導覽列按鈕實現無縫切換。
2. **極致手機版優化 (RWD)**：使用 CSS Grid 網格佈局，卡片在電腦版自動排成三欄，在手機版則自動適應為單欄，滑動流暢不跑版。
3. **一鍵跳轉地圖**：每個美食卡片皆標記詳細地址與營業時間，並利用 `<a>` 標籤的 `target="_blank"` 屬性，讓使用者點擊時能直接開啟新分頁跳轉至 Google 地圖進行導航。
4. **潮流視覺設計**：放棄傳統網頁的死板排版，採用高飽和度的活力橘、初戀粉與酪梨綠作為各城市主題色，並搭配斜切（clip-path）設計與卡片懸浮（hover）特效。

## 🤖 AI 協作與難關突破說明

在專案開發過程中，我們透過與 AI 工具（Gemini）深度對話來協助編碼，並手動克服了以下三大真實技術痛點：
* **地圖跳轉失效**：初次輸入地圖連結時因語法衝突無法點擊，後經團隊手動調整 URL 格式並補上 `target="_blank"` 成功修復。
* **GitHub 圖片破圖**：上傳至 GitHub Pages 後發現部分照片無法顯示。團隊自主檢查後發現 GitHub 對英文大小寫與相對路徑要求極為嚴格，手動將所有圖檔命名規範化後解決。
* **提示詞溝通失焦**：初期因指令過於抽象導致 AI 產出不符合預期。後期團隊調整策略，改給予具體色碼與 Grid 佈局參數，成功實現精準協作。
